# draw. guess. repeat.

Squiggle is a lightweight, open-source multiplayer drawing game built with **Next.js 15** and **Firebase**. One player draws, the rest guess — with real-time canvas sync, live chat, emoji reactions, and a server-side word bank that keeps cheating impossible.

## Key features
