# draw. guess. repeat.

Squiggle is a lightweight, open-source multiplayer drawing game. One player draws, the rest guess — with real-time canvas sync, live chat, emoji reactions, and a server-side word bank that keeps cheating impossible.

## What is Squiggle?

Squiggle is a real-time multiplayer drawing and guessing game built with **Next.js 15**, **Firebase Firestore**, and **TypeScript**. It's designed to be:

- **Private** — rooms use 6-character codes; no public lobbies
- **Cheat-proof** — the word bank lives entirely in Firestore, never in the client bundle
- **Configurable** — hosts control rounds, draw time, hints, categories, and more
- **Lightweight** — no heavy game server; all real-time state syncs through Firestore
